# AuraWed AI婚礼策划平台 - 需求规格说明书

---

## 一、平台定位

**B/C 双端 AI 策划服务平台**

- B端：普通用户（提交需求）
- C端：策划师（接单、交付方案）

**核心流程**：用户填写需求 → AI生成故事 → AI生成方案 → AI生成设计图 → 派单给策划师

**技术路线**：Next.js + Supabase + AI API（多模型接入）

---

## 二、账号体系

### 1. 用户账号（B端）

**功能**：
- 注册、登录、退出
- 个人中心
- 查看我的订单/需求
- 查看AI生成的内容：美学小故事、策划方案、设计图
- 在线支付购买服务
- 联系客服/管理员
- 修改昵称、头像、密码

**禁止**：积分、等级、签到、社交功能

### 2. 策划师账号（C端）

**功能**：
- 注册、申请入驻（等待管理员审核）
- 登录、个人中心
- 查看可接单的需求列表
- 接单 / 拒单
- 上传最终方案、文件
- 查看自己的订单、收入统计
- 联系管理员

**禁止**：自动提现、自动分账、店铺装修

### 3. 管理员账号

**功能**：

1. **用户管理**
   - 查看所有用户列表
   - 禁用/解封异常用户

2. **策划师管理**
   - 审核策划师入驻
   - 设置策划师是否可接单
   - 查看策划师收入与订单

3. **订单/需求管理（核心）**
   - 查看所有用户提交的需求
   - 查看AI是否生成成功
   - 手动派单给策划师
   - 重发/重新生成AI内容

4. **内容管理**
   - 查看所有AI生成的故事、方案、图片
   - 可编辑、补发、删除

5. **支付与流水**
   - 查看所有支付订单
   - 查看金额、支付状态

6. **客服/公告**
   - 发布网站公告
   - 查看用户留言/反馈

7. **简单数据统计**
   - 今日订单数、总订单、付费人数
   - API调用次数（简单展示）

8. **系统设置**
   - 修改网站名称、LOGO、客服联系方式
   - 设置价格（基础版/高级版）

---

## 三、核心业务流程

### 用户端流程

1. **填写表单**
   - 场景（婚礼/活动/生日等）
   - 风格偏好
   - 预算
   - 喜欢元素
   - 期望氛围

2. **提交 → 支付**

3. **AI生成（3步）**
   - 第一步：生成美学小故事（挖掘内心需求）
   - 第二步：生成完整策划方案
   - 第三步：调用文生图API生成设计图

4. **生成完成 → 用户在个人中心查看**

5. **管理员派单 → 策划师接单 → 人工优化 → 交付最终版**

### 策划师流程

1. 申请入驻 → 管理员审核
2. 查看平台派来的订单
3. 接单 → 查看用户需求 + AI初稿
4. 上传最终方案
5. 完成订单 → 等待管理员结算

### 管理员流程

1. 审核策划师
2. 查看订单状态
3. 手动派单
4. 处理异常订单（API失败、用户投诉）
5. 手动给策划师结算

## 四、API与稳定性要求

### 1. 多模型API接入
- 主模型 + 兜底模型
- 可在后台切换

### 2. 超时机制
- 单个API请求超时8-10秒

### 3. 自动重试
- 失败自动重试2次

### 4. 失败兜底
- API全部失败时，返回预设文案 + 缓存方案
- 不显示技术报错

### 5. API Key安全
- Key存在后端，不暴露在前端
- 防止被盗刷

### 6. 限流
- 同一用户1分钟内最多提交2次需求
- 防止恶意刷量

## 五、前端页面要求

### 必须页面

- 首页
- 登录/注册
- 用户需求提交表单页
- 支付页
- 个人中心（用户）
- 策划师中心

### 交互要求

- 生成中显示友好加载动画
- 不卡顿、不空白
- 错误只提示用户易懂文字
- 适配手机端

## 七、本地数据支撑

### 设计库
- 10+ 配色方案
- 8 种风格方向
- 6 种花艺风格
- 5 种灯光设计

### 策划库
- 4 种预算模板
- 3 种流程模板
- 7 种区域模板
- 2 种服务套餐

### 故事库（兜底）
- 8 种故事模板（校园、职场、旅行等）
- 可随机组合生成更多
- API失效时自动切换本地方案

## 八、技术栈

- **前端**：Next.js + React + Tailwind CSS
- **后端**：Next.js API Routes
- **数据库**：Supabase（PostgreSQL）
- **AI**：Google Gemini（文字+图片）
- **部署**：Vercel / 独立服务器
